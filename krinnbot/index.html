GENERAL COMMANDS


  

    command
    function
  

  

    -c
    displays basic help menu.
  

  

    -ping
    checks ping.
  

  

    -say / -repeat
    says whatever comes after the command.
  

  

    -info
    displays server info.
  

  

    -botinfo
    displays bot info.
  

  

    -8ball
    generates a random response to a question.
  

  

    -suggest
    links to AJLStick's e-mail for suggestions, will be improved upon at a later date.
  

  

    -nword
    generates a random word starting with n.
  

  

    -ins
    generates a random insult string.
  

  

    -arg / -coin
    flips a coin.
  

  

    -secret
    sends a timed message to the channel and a secret hint to the sender's DMs.
  



ECONOMY AND ADMIN COMMANDS

  

    -minutely

    -hourly
    recieve your hourly funds. [1,500]
  

  

    -daily

    -lvlup
    spend 5,000 to level up one time.
